# Phase 04 — Calendar v1: week/day/month + plan vs done

## Goal
Usable calendar with plan vs done cards and post-training feedback drawer.

## Scope
- Week, Day, Month views
- Planned workouts (ghost style) vs Completed activities (solid style)
- Navigation (prev/next, today)
- Post-training feedback drawer:
  - RPE (1–10)
  - feel chips
  - comment
  - soreness/pain flag
- Mobile month: 1–2 items + "+X more" + day detail panel
- In-memory mock data only
